Flight App

This is a JavaScript project with a few event listeners and functions. It makes requests to an API to collect flight data based on user inputs. The project also contains some functions to manipulate the DOM and display the results of the user's search.

Movies App

The code defines a constructor function Movie that takes in several properties for a movie, such as name, length in minutes, category, and price. It then creates an HTML element for the movie, including an image and a delete button. It also creates an array of movie objects, and displays them on the webpage, along with a search bar to filter the movies and a form to add new movies. The code also includes functionality for filtering movies by length, sorting movies by various properties such as category and price, and deleting movies from the array and adding them to an archive array. Overall, the code is a simple web application for managing a collection of movies.

CryptoCurrency App

This project is a web page that displays data on cryptocurrencies from the Binance API. It uses JavaScript to fetch data from the API and create an array of Exchange objects with the fetched data. The page displays a table of the data, and allows users to filter the data by currency name, price range, and volume range. It also has a feature to display the top 10 currencies by volume.

Car Game

The game was built using JavaScript classes, HTML grid layout, and CSS for styling. The Car class creates a car object with properties such as direction, location, and rotation. The Target and Bomb classes create objects for the randomly generated targets and bombs, respectively. The game includes functions for handling key events, detecting collisions between the car and targets/bombs, and updating the score display. The game also features functions for preventing the targets and bombs from occupying the same location on the grid.

Website bookmarks

This is a simple web application that allows users to add and save bookmarks for their favorite websites. The application is built using HTML, CSS, and JavaScript and uses Bootstrap and Font Awesome for styling and icons. It also makes use of LocalStorage to store the saved bookmarks in the browser, making them available even after the user closes the browser or refreshes the page.

Timer-app

This project is a simple web-based timer application built using HTML, CSS, and JavaScript. It allows users to set a countdown timer by entering minutes and seconds, along with a description of the timer. The application includes features such as starting, stopping, and deleting timers. The timer display updates in real-time and includes a visual representation of the remaining time. The application uses event listeners and DOM manipulation to dynamically update the interface based on user input.

Car game

This project is a simple browser-based game where the player controls a car using arrow keys to catch targets and avoid bombs. The game ends if the car hits a bomb. The player's score is updated each time they catch a target, and a modal window pops up displaying the final score after the game ends.

Bookmark Page

This project is a bookmark manager that allows users to save and organize their favorite websites. It uses JavaScript and LocalStorage to store and retrieve user data, and dynamically generates HTML elements to display the bookmarks. The user can add, delete, and clear bookmarks, and the interface is designed to be simple and user-friendly.

Timer Page

This project is a simple timer application built using JavaScript and HTML/CSS. It allows users to set a timer by specifying the number of minutes and seconds, give a description for the timer, and then start, stop, and delete the timer. The timer is displayed on the page in a container along with buttons to control it. The code includes functions for creating and formatting the timer, as well as for starting, stopping, and deleting it. It also includes input validation to ensure that the user enters valid values for minutes and seconds.
